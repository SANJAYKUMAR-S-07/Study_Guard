Study Guard Console

Because VS Code blocks camera access in sidebar tabs due to security guidelines, Study Guard runs the camera engine inside your web browser.

Local Service Status
Active (Port {{PORT}})
How to Use
  1. Click the Open Tracker in Browser button above.
  2. Grant camera permissions to localhost in your browser.
  3. Adjust trigger delay and sensitivity sliders on the browser console.
  4. Leave the browser page open in the background (or split-screen) and start coding in VS Code!
